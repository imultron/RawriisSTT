  Debian/Ubuntu:  sudo apt install python3-pyaudio
  Arch/Manjaro:   sudo pacman -S python-pyaudio
  Fedora:         sudo dnf install python3-pyaudio
Other STT engines (Whisper, Vosk, Azure) do not require PyAudio.
